# llm-challenge

Joke chatbot grounded on 100 jokes fetched from JokeAPI. Built with Flask + ChromaDB + OpenRouter (Gemini).

The bot only answers questions about its joke database — it won't generate jokes, answer off-topic questions, or serve NSFW content.

## Stack

- Flask (web server)
- ChromaDB (local vector store)
- sentence-transformers `all-MiniLM-L6-v2` (local embeddings)
- OpenRouter → `google/gemini-2.0-flash-001` (LLM)

## Setup

**1. Install dependencies**

```bash
pip install -r requirements.txt
```

Or with uv:

```bash
uv venv
source .venv/Scripts/activate  # Windows
uv pip install -r requirements.txt
```

**2. Set up your .env**

```bash
cp .env.example .env
# add your OPENROUTER_API_KEY
```

**3. Fetch the jokes**

```bash
python fetch_data.py
```

This hits JokeAPI and saves 100 jokes to `data/jokes.json`. Only needs to run once.

**4. Ingest into ChromaDB**

```bash
python ingest.py
```

Embeds all 100 jokes and stores them locally. Run again if you change the data.

**5. Start the app**

```bash
python app.py
```

Open http://localhost:5000

## Docker

```bash
docker compose up --build
```

Note: run `python ingest.py` first so the `chroma_db/` folder exists before Docker starts.

## Evaluation

```bash
python eval.py
python eval.py --verbose
python eval.py --save
```

## Project structure

```
app.py          - Flask routes
rag.py          - retrieval + generation pipeline
ingest.py       - embeds jokes.json into ChromaDB
fetch_data.py   - fetches jokes from JokeAPI
eval.py         - test harness
data/           - jokes.json lives here
templates/      - chat UI
chroma_db/      - generated by ingest.py, gitignored
```
